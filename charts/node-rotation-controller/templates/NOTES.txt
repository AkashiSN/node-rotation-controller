   per NodePool group you want rotated (spec §5.4):

     kind: RotationPolicy
     metadata:
       name: api
     spec:
       nodePoolSelector:
         matchLabels:
           workload: api
       maintenanceWindows:
         - timezone: Asia/Tokyo
           days: [Wed, Sat]
           start: "02:00"
           end: "06:00"
     EOF

   A NodePool matched by no RotationPolicy is not rotated (its expireAfter
